# WorkRamp

WorkRamp is an AI-first revenue enablement platform providing a REST API for managing learning paths, assessments, competencies, coaching, and onboarding programs for go-to-market teams. It serves both internal employee training (Employee Learning Cloud) and external customer/partner education (Customer Learning Cloud).

## API

- **Base URL:** `https://app.workramp.com/api/v1`
- **Auth:** Bearer token (API key generated in Settings → Integrations → API)
- **Docs:** https://developers.workramp.com/reference/getting-started
